# Fruit Grammar Ninja 🍉

A fun fruit-slicing game for middle school students to practise **verbs vs nouns** in English.

## How to Play

- Fruits with English words fall from the top
- The game tells you to slice **VERBS** or **NOUNS**
- Click/tap the correct fruits to score points
- ❌ Wrong slice = lose a life
- 🔥 Combo = bonus points

## Play Online

Open `index.html` in any modern browser. No server or installation needed.
